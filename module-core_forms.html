(inner) sanitizeString(string, maxLength) → {string}

Sanitizes a string form input (by removing any tags and slicing it to the max allowed size). If the string is meant for anything other than direct display (e.g. links, markdown...) then this is not the right filter. Use this on all string input unless you need more advanced escaping (e.g. for URLs, for Markdown)
